
# Product Requirements Prompt (PRP) — [Status Marker] ✅ Completed

## 1. Project Context & Objectives

The current UI layout clips on mobile devices, and the Master Volume fader suffers from automation-loop latency and incorrect perceptual scaling. This phase tightens the vertical layout, implements standard overflow scrolling, relocates the settings cog, and refactors the global volume mapping to update instantaneously via logarithmic amplitude scaling.

## 2. Technical Decisions & Dependencies

* **Logarithmic Volume:** The UI slider will be converted to a `0-100` range. The `GenerativeAudioController` will map this to Tone.js using standard amplitude-to-decibel conversion.
* **Instant Responsiveness:** The volume will be removed from the 1-second `Tone.Transport` loop and instead trigger a 0.1s `rampTo` immediately upon payload reception.
* **Responsive CSS:** The application's root constraints (`h-screen`, `overflow-hidden`) will be relaxed to `min-h-screen`, `overflow-y-auto`, and `pb-24` to accommodate the physical Safari toolbar.

### Phase 3: Zero-Latency Logarithmic Engine Mapping — [Status Marker] ✅ Completed

* **Objective:** Remove the volume from the 1-second automation tick and apply it instantaneously using amplitude-to-decibel math.
* **Instructions:**
1. Open `src/engine/GenerativeAudioController.js`.
2. Locate `processAutomationTick` and **delete** the block that calculates and applies `globalVolume` to `Tone.Destination` (the code we added in Phase 5 of the last PRP).
3. Locate the `updatePayload(payload)` function at the top of the class. Update it to immediately process the volume calculation:
```javascript
updatePayload(payload = {}) {
  this.currentPayload = { ...this.currentPayload, ...payload };

  if (payload.mixerState) {
    this.applyMixerState(payload.mixerState);
  }

  // Instant, Logarithmic Master Volume
  if (payload.masterVolume !== undefined) {
    const uiVal = payload.masterVolume;
    const dest = Tone.getDestination ? Tone.getDestination() : Tone.Destination;

    if (dest && dest.volume && typeof dest.volume.rampTo === 'function') {
      if (uiVal <= 0) {
        dest.volume.rampTo(-Infinity, 0.1);
      } else {
        // Map 0-100 linearly to amplitude, converted to Decibels
        const db = 20 * Math.log10(uiVal / 100);
        dest.volume.rampTo(db, 0.1); // Fast 100ms response
      }
    }
  }
}

```

* **Test Case 2:** [Given `updatePayload` is called with `masterVolume: 0`, When executed, Then `Tone.Destination.volume` ramps to `-Infinity`.]
* **Status Update:** "Transition Phase 3 status to 🔵 Active. Apply modifications. Once complete, update to ✅ Completed."
