# Product Requirements Prompt (PRP) — [Status Marker] ✅

## 1. Project Context & Objectives
The system currently hardcodes pitch generation and binaural entrainment to static values. This PRP introduces a global "Key Root" selector in the settings, allowing dynamic transposition. The generative loops will calculate pentatonic notes on the fly using MIDI offsets from the selected root, and the binaural oscillators will perfectly match the selected root's standard exact frequency (Hz).

## 2. Technical Decisions & Dependencies
* **Standard Frequency Map:** A constant dictionary (`TONE_KEY_MAP`) will map keys (C through B) to their respective Octave 3 MIDI numbers and exact standard Hz frequencies.
* **MIDI Math:** The `PitchMarkovEngine` outputs structural degrees (`1, 2, 3, 5, 6`). We will map these to standard Major Pentatonic MIDI intervals (`0, 2, 4, 7, 9`) and add them to the root MIDI number, utilizing `midiToNote()` to translate back to Tone.js string formats.
* **Detune Removal:** Because the binaural beats and music will now share standard equal-temperament frequencies natively, the `detune: 35` property previously added to the synths must be removed.

        ```javascript
        const TONE_KEY_MAP = {
          'C': { rootMidi: 48, freq: 130.81 },
          'C#': { rootMidi: 49, freq: 138.59 },
          'D': { rootMidi: 50, freq: 146.83 },
          'D#': { rootMidi: 51, freq: 155.56 },
          'E': { rootMidi: 52, freq: 164.81 },
          'F': { rootMidi: 53, freq: 174.61 },
          'F#': { rootMidi: 54, freq: 185.00 },
          'G': { rootMidi: 55, freq: 196.00 },
          'G#': { rootMidi: 56, freq: 207.65 },
          'A': { rootMidi: 57, freq: 220.00 },
          'A#': { rootMidi: 58, freq: 233.08 },
          'B': { rootMidi: 59, freq: 246.94 },
        };
        const PENTATONIC_OFFSETS = { 1: 0, 2: 2, 3: 4, 5: 7, 6: 9 };
        ```
    4.  Locate `this.layer2Synth` and `this.layer3Synth` in `bootEngine()`. **Delete** the `detune: 35` lines.
    5.  Locate `this.harmonicLoop` and `this.melodicLoop`. Update their callbacks to compute pitch dynamically:
        ```javascript
        this.harmonicLoop = new Tone.Loop((time) => {
          if (Math.random() < 0.40) {
            const currentSecs = Tone.getTransport ? Tone.getTransport().seconds : Tone.Transport.seconds || 0;
            const degree = this.pitchEngine.generateNextDegree({
              currentTimeMinutes: currentSecs / 60,
              solTargetMinutes: this.currentPayload.solTarget || 20
            });
            const key = this.currentPayload.musicalKey || 'D';
            const rootMidi = TONE_KEY_MAP[key]?.rootMidi || 50;
            const pitch = midiToNote(rootMidi + PENTATONIC_OFFSETS[degree]);
            this.layer2Synth.triggerAttackRelease(pitch, "2n", time, 0.2);
          }
        }, "2n");

        this.melodicLoop = new Tone.Loop((time) => {
          if (Math.random() < 0.30) {
            const currentSecs = Tone.getTransport ? Tone.getTransport().seconds : Tone.Transport.seconds || 0;
            const degree = this.pitchEngine.generateNextDegree({
              currentTimeMinutes: currentSecs / 60,
              solTargetMinutes: this.currentPayload.solTarget || 20
            });
            const key = this.currentPayload.musicalKey || 'D';
            const rootMidi = TONE_KEY_MAP[key]?.rootMidi || 50;
            // Melodic layer plays one octave higher (+12)
            const pitch = midiToNote(rootMidi + PENTATONIC_OFFSETS[degree] + 12);
            this.layer3Synth.triggerAttackRelease(pitch, "4n", time, 0.15);
          }
        }, "4n");
        ```
