# Product Requirements Prompt (PRP) — 🔘 Pending

## 1. Project Context & Objectives
The cross-context contamination between Tone.js and the native `AudioWorkletNode` has been successfully resolved by injecting a native hardware context. However, this raw injection bypassed the initialization of Tone.js's internal Web Worker clock (`Tone.Clock`). As a result, `Tone.Transport` is frozen, causing the `Tone.Loop` schedulers for the Harmonic and Melodic synth layers (Layers 2 and 3) to remain silent, while the immediate-trigger Drone layer (Layer 1) plays correctly.

**Objective:** Wrap the natively injected `AudioContext` within the official `Tone.Context` class before binding it. This guarantees that Tone.js spins up the Web Worker necessary to drive the `Tone.Transport` timeline, restoring playback to all sequenced loops.

## 2. Technical Decisions & Dependencies
* **Context Wrapper:** We will modify the injection in `PlayButton.jsx` to use `new Tone.Context(nativeCtx)`.
* **Transport State Alignment:** We will add a strict `Tone.Transport.cancel(0)` cleanup phase inside `startSession()` to prevent orphaned loops from previous renders from clogging the newly initialized timeline.

## 3. Implementation Phases

### Phase 1: Clock Worker Revival (PlayButton.jsx) — ✅ Completed
* **Objective:** Ensure Tone.js boots its clock worker when receiving the custom context.
* **Instructions:**
    1. Open `src/components/PlayButton.jsx`.
    2. Locate the injection logic inside `handlePlayPauseClick`.
    3. Update the injection block to instantiate a `Tone.Context` explicitly:
        ```javascript
        if (!window.somnusContextInjected && typeof window !== 'undefined') {
          const NativeAudioContext = window.AudioContext || window.webkitAudioContext;
          if (NativeAudioContext) {
            const nativeCtx = new NativeAudioContext();
            
            // Wrap the native context to force Tone to boot its internal Web Worker Clock
            const toneContext = new Tone.Context(nativeCtx);
            Tone.setContext(toneContext);
          }
          window.somnusContextInjected = true;
        }
        ```

### Phase 2: Transport Timeline Alignment (GenerativeAudioController.js) — ✅ Completed
