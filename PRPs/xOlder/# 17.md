# Phased Development Plan — [Status Marker] 🔘

## 1. Project Context & Strict Constraints

Somnus v1.5 temporarily pauses active development on the Generative Music Engine to resolve spatial orientation artifacts and low-frequency stagnation. The pivot establishes a highly stable, 5-channel Acoustic Entrainment Subsystem. This subsystem will actively downregulate the sympathetic nervous system using binaural beats, isochronic pulses, masking noise, and nature ambience. The transition logic is strictly governed by a 3-Phase Lifecycle (Descent, Hold, Fade) and a dynamic Master Low-Pass Filter (LPF) that simulates the acoustic dampening of a quiet physical environment.

## 2. Enforced Dependencies & Architecture

* **Audio Engine:** Tone.js (v15.0.4) strictly utilizing `Tone.Oscillator` (Sine), `Tone.Noise`, `Tone.Tremolo` (or LFO -> Gain), `Tone.Player`, `Tone.CrossFade`, and `Tone.Filter`.
* **Framework:** React (v18.3.1) with Context API for state management.
* **Testing:** Vitest (v3.0.4) and JSDOM.
* **Routing Architecture:** 5 distinct Channels -> Master LPF (-24dB/octave) -> Master Fade -> Output.
* **Asset Paths:** Nature Ambience buffers must load from `/Nature Sounds Audio/Rain/` and `/Nature Sounds Audio/Ocean Waves/`.
