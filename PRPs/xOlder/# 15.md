## 1. Project Context & Objectives
The native audio graph is now fully connected and bridging Tone.js into the custom `fdn-reverb-worklet` successfully (confirmed by binaural audio output). However, the Worklet is currently outputting a dry, direct signal without the required diffuse Reverb tail. Additionally, the user is only perceiving 2 of the 3 synthesis layers, likely due to extreme volume attenuation on Layer 3 and the lack of a reverberant field to carry the higher frequencies.

**Objective:** Force the `AudioWorkletNode` into a 100% wet, "massive far-field" state by programmatically maximizing its Reverb parameters immediately after instantiation. Secondly, re-balance the gain staging of the three synth layers to ensure equal perceptual presence.

## 2. Technical Decisions & Dependencies
* **Dynamic Parameter Initialization:** Since the exact parameter names of the custom WASM/Worklet are encapsulated, the agent will iterate through the `AudioWorkletNode.parameters` map and programmatically max out anything related to 'wet', 'mix', 'decay', 'size', or 'diffusion'.
* **Gain Staging:** Normalize the layer volumes to a tighter grouping (-8dB, -10dB, -10dB) to ensure Layer 3 is not psychoacoustically masked.

## 3. Implementation Phases

### Phase 1: Worklet Parameter Maximization (GenerativeAudioController.js) — ✅ Completed
* **Objective:** Ensure the FDN Reverb operates in a 100% diffuse, far-field state.
* **Instructions:**
    1. Open `src/engine/GenerativeAudioController.js`.
    2. Locate the `bootEngine()` method, specifically inside the `try` block where `this.reverbWorklet = new window.AudioWorkletNode(...)` is instantiated.
    3. Immediately *after* creating the `this.reverbWorklet` and connecting the native routing, insert this dynamic parameter loop:
        ```javascript
        // Force Worklet into Massive Far-Field / 100% Diffuse State
        if (this.reverbWorklet.parameters) {
          this.reverbWorklet.parameters.forEach((param, name) => {
            const paramName = name.toLowerCase();
            // Maximize Wet/Mix parameters
            if (paramName.includes('wet') || paramName.includes('mix')) {
              param.value = param.maxValue !== undefined ? param.maxValue : 1.0;
            }
            // Maximize Room Size / Decay / Diffusion
            if (paramName.includes('size') || paramName.includes('decay') || paramName.includes('diffuse') || paramName.includes('time')) {
              param.value = param.maxValue !== undefined ? param.maxValue : 10.0;
            }
            // Minimize Dry signal
            if (paramName.includes('dry')) {
              param.value = param.minValue !== undefined ? param.minValue : 0.0;
            }
          });
        }
        ```
