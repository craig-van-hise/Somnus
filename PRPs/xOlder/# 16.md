# Product Requirements Prompt (PRP) — 🔘 Pending

## 1. Project Context & Objectives
The application is currently playing audio through the audio graph, but two critical neurological design violations are present:
1. **Localization & Proximity:** The audio feels "close" and in-room rather than massive, diffuse, and far-field, because the custom 2OA FDN Reverb `AudioWorkletNode` parameters are not being correctly configured or mapped upon initialization.
2. **Pseudo-Panning / Spatial Movement:** The sources are perceived as moving or panning around. This directly violates the core operating principles of Somnus: spatial movement triggers the brain's orienting reflex and cortical hyperarousal, preventing Slow-Wave Sleep (SWS) induction. All sources must remain completely static, diffuse, and centered/immersed in a wide acoustic field.

**Objective:** Configure the `AudioWorkletNode` parameters correctly using its `parameters` port or message port interface, locking the reverb into a maximum-decay, ultra-diffuse, far-field state, and ensure no panning or stereo modulation exists anywhere in the synth routing chain.

## 2. Technical Decisions & Dependencies
* **Worklet Message/Parameter Binding:** Unlike standard native AudioParams, custom AudioWorkletNodes often handle internal configuration via `workletNode.port.postMessage()` or specific parameter IDs defined in `fdn-reverb-worklet.js`. We will inspect/configure both standard AudioParams and message passing to ensure the reverb decay and wet/dry mix are locked to 100% diffuse far-field.
* **Zero-Panning Enforcement:** Audit all synth routes to ensure strict monophonic/static stereo balancing with zero LFO panners, stereo spreaders, or random pan calculators.

        ```javascript
        if (this.reverbWorklet) {
          // 1. Try standard AudioParam iteration if available
          if (this.reverbWorklet.parameters) {
            for (let [name, param] of this.reverbWorklet.parameters.entries()) {
              const lower = name.toLowerCase();
              if (lower.includes('wet') || lower.includes('mix')) {
                param.value = 1.0; // 100% wet diffuse tail
              } else if (lower.includes('decay') || lower.includes('size') || lower.includes('time') || lower.includes('room')) {
                param.value = param.maxValue !== undefined ? param.maxValue : 20.0; // Maximum far-field decay
              } else if (lower.includes('dry')) {
                param.value = 0.0; // Zero dry direct sound
              }
            }
          }

          // 2. Send initialization configuration message via MessagePort (standard for custom worklets)
          if (this.reverbWorklet.port && typeof this.reverbWorklet.port.postMessage === 'function') {
            this.reverbWorklet.port.postMessage({
              type: 'SET_PARAMS',
              config: {
                wet: 1.0,
                dry: 0.0,
                decayTime: 15.0, // Massive far-field decay
                diffusion: 0.95,
                roomSize: 1.0
              }
            });
          }
        }
        ```
