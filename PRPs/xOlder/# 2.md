The Developer Console is throwing this error when initializing the Reverb Worklet: TypeError: Failed to construct 'AudioWorkletNode': parameter 1 is not of type 'BaseAudioContext'.

You are currently passing the Tone.js context wrapper (Tone.context) into the native AudioWorkletNode constructor. You must pass the raw native audio context instead.

Update the instantiation in GenerativeAudioController.js from new AudioWorkletNode(Tone.context, ...) to new AudioWorkletNode(Tone.getContext().rawContext, ...). Ensure that when you route the Layer outputs to this Worklet, you use Tone.connect(sourceNode, workletNode) to successfully bridge the Tone.js nodes to the native Web Audio API node.
